# 💊 Pharmacy Network & Prescription Management System

## 📌 Overview

A full-stack web application that connects users with multiple pharmacies on a single platform. Users can search medicines, compare prices, and place orders, while pharmacies manage inventory and handle requests.

---

## 🚀 Features

### 👤 Patient

* Search medicines
* Compare prices across pharmacies
* Place orders
* Track order status

### 🏪 Pharmacy

* Add medicines (name, price, stock)
* Manage inventory
* Accept / Reject orders

---

## 🧠 Key Highlights

* Multi-pharmacy (multi-vendor) system
* Real-time medicine availability
* Price comparison engine
* Order lifecycle management

---

## 🏗️ Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB

---

## ⚙️ Setup Instructions

### 1. Clone Repo

```
git clone https://github.com/your-username/pharmacy-network.git
cd pharmacy-network
```

---

### 2. Backend Setup

```
cd backend
npm install
node server.js
```

---

### 3. Frontend Setup

```
cd frontend
npm install
npm start
```

---

### 4. Run MongoDB

Make sure MongoDB is running locally or use MongoDB Atlas.

---

## 🌐 API Endpoints

* `POST /add-medicine` → Add medicine
* `GET /medicines` → Get all medicines
